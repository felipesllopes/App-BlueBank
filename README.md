# App-BlueBank

## My bank app (app in construction) ⚠

### Description

This application is a banking app simulator, where it will be possible to carry out transactions, and some ATM transactions, such as withdrawals and deposits, as a practice of changing the value in the database.
To connect to the bank, you need to have an account, or open one. Through the registered email and password, the bank will be authenticated using the firebase authentication method.
