# App-BlueBank

## My bank app (app in construction) ⚠

### Description

This application is a banking app simulator, where it will be possible to carry out transactions, and some ATM transactions, such as withdrawals and deposits, as a practice of changing the value in the database.
To connect to the bank, you need to have an account, or open one. Through the registered email and password, the bank will be authenticated using the firebase authentication method.

### Loot screen

Screen that simulates a withdrawal, where it goes through several checks to find out if the user can withdraw a certain amount. If there is not enough balance to carry out the withdrawal, a message is issued to the user. If the withdrawal is allowed, a screen confirming the withdrawal of the determined amount is opened.

![tela4](https://github.com/felipesllopes/App-BlueBank/assets/99768939/2f94a8fb-7810-4be2-a3ad-de9b162bd389)

### Transaction history

Transaction history screen, where the user can consult his history and verify transactions carried out. The feature of choosing a date will be added, so that the user can consult the history of a given day.
